Site Name

LivingHistory will be the site name as users can use FamilySearch data to chat with an ai character and see images based on a deceased ancestor from FamilySearch.

Site Purpose

The purpose of this website is to allow a user to login to FamilySearch, and see image memories attached to a deceased ancestor. Users can also chat with an AI character based on information about that deceased ancestor.

Scenarios

How do I log into FamilySearch?

There will be a button labeled "Login to FamilySearch" to begin the login process with FamilySearch. You will have to give permission for this website to access data from FamilySearch about your deceased ancestors.

How do I select a deceased ancestor?

A drop down menu will be presented for you with a list of 4 generations of deceased ancestors for you to view images attached to, and to setup an AI character with a personality drawn from data about that ancestor on FamilySearch.
